# FileGuard-A-GUI-Based-File-Safety-Utility-Conversion-System
DESCRIPTION :- File Guard aims to offer a secure environment for file uploads and file conversions.
The application first scans the file for viruses or malicious patterns by using an antivirus engine. 
If the file is clean, then the user can proceed with various file conversion operations.
OBJECTIVES :- To develop a GUI-based application that allows users to safely upload and convert files.
To ensure file safety using malware/virus scanning before processing.
To implement file type conversions (document and image formats) through automated tools.
Functional Requirements: A GUI interface for selecting and uploading files.
Ability to scan uploaded files for viruses or malicious content.
File conversion support (e.g., DOCX → PDF, JPG → PNG, etc.).
Option to download the converted and safe file.
Logging of all file operations and results.
CONCLUSION :- File Guard successfully integrates file safety and file conversion in a single application. The use of a GUI makes it accessible for end-users, while the integration of antivirus scanning ensures security.
